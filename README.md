# Ítalo Botelho - Portfólio 2026

Bem-vindo ao repositório do meu portfólio pessoal! Desenvolvido para apresentar meus projetos na área de Ciência de Dados, Inteligência Artificial e Desenvolvimento de Sistemas.

## 🛠 Tecnologias

- **[Next.js](https://nextjs.org/)** (App Router)
- **[React](https://reactjs.org/)**
- **[Tailwind CSS](https://tailwindcss.com/)**
- **[Framer Motion](https://www.framer.com/motion/)** para animações
- **[Three.js](https://threejs.org/)** para o fundo 3D interativo

## 🚀 Como rodar o projeto localmente

1. Faça o clone do repositório:
```bash
git clone https://github.com/italobotelho/portfolio-2026.git
```

2. Entre na pasta do projeto e instale as dependências:
```bash
cd portfolio-2026
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.
